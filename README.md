<h1>BookManager (DigitalBookViewer)</h1>

<h2>Description</h2>
<p>
Application <strong>Windows Forms</strong> permettant de charger un livre
sous forme de fichiers <code>.txt</code> et de naviguer entre les pages.
</p>

<h2>Technologies</h2>
<ul>
  <li>C#</li>
  <li>Windows Forms</li>
  <li>.NET Framework 4.7.2</li>
</ul>

<h2>Fonctionnalités</h2>
<ul>
  <li>Chargement d’un dossier contenant des pages <code>.txt</code></li>
  <li>Navigation entre les pages</li>
  <li>Accès direct à une page précise</li>
</ul>

<h2>Lancer le projet</h2>
<ol>
  <li>Ouvrir le fichier <code>BookManager.sln</code> avec <strong>Visual Studio</strong></li>
  <li>Compiler le projet (Build)</li>
  <li>Lancer l’application (Run)</li>
  <li>Entrer le chemin du dossier contenant les pages</li>
</ol>
